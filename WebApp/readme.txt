The “search interesting events in Carleton“ webapp
CS257 Software Design, Spring 2016
Yanhan Lyu, Lucy Wu

==== Run this app on your own computer ====


1. Do NOT use Safari to open the web for it would not support html format of “input”. Use Chrome.

2. Allow pop-ups in Chrome in order to enable Google calendar page to show up.

3. It might take a few seconds for events to show up. It does not often act like that, but there is a fairly high chance that it would happen.
